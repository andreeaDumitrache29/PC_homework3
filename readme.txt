Task 1
	Declaram o structura color cu campuri r,g,b de tip unsigned char pentru a stoca culoarea unui pixel. Se deschide apoi fisierul de input si citim din el datele de intrare necesare taskului: numele imaginii ce trebuie modificata si noua culoare a cifrelor. Deschidem apoi fisierul binar asociat imaginii si citim din el cele doua headere:bmp_fileheader si bmp_infoheader, dupa care acestea sunt scrise in fisierul f, unde se va afla noua imagine.
	Parcurgem imaginea (matrice de tipul structurii color) pixel cu pixel si verificam daca pixelul curent este alb (adica toate cele 3 canale ,r g, b, sunt 255): daca da, il scriem asa cum este in noua imagine, iar daca nu ii modificam culoarea conform cerintei. Calculam padding-ul matricei de pixeli pentru a-l sari la citire si pentru a-l intializa cu 0 in noua matrice.Aceasta se scrie pas cu pas intr-un nou fisier cu numele specificat.

Task 2
	Declaram o structura color cu campuri r,g,b de tip unsigned char pentru a stoca culoarea unui pixel. Functia verify verifica daca o matrice b data ca parametru face match pe o portiune de 5x5 pixeli dintr-o matrice mai mare: functia returneaza 1 in caz afirmativ, si 0 in caz negativ.
	Se deschide fisierul de input si citim din el datele de intrare necesare taskului, respectiv numele imaginii.
	Deoarece toate cifrele sunt matrici cu diemnsiunea cunoscuta si constanta declaram cate o matrice cu elemente de tip char pentru fiecare dintre cele 10 cifre. Construim apoi o copie a matricei initala de pixeli, insa cu elemente de tip char, in care un pixel alb va fi reprezentat prin '.', iar un pixel colorat prin 'x. Cautam apoi in noua matrice de caractere cate un pixel colorat ('x') si verificam cu ajutorul functiei verify cu care dintre matricele de caractere face match. Dupa ce am gasit cifra corespunzatoare o punem intr-un vector de cifre, iar elementele de tip 'x' le transformam in '.' pentru a nu fi gasite din nou la urmatoarea parcurgere a matricei.
	In final scriem cifrele stocate in vectorul "cifra" in fisierul text cu nume corespunzator.

Bonus
	Functia dif returneaza numarul de pixeli diferenta dintre o matrice de 5x5 data ca parametru si o portiune de 5x5 din matricea principala.
	In functia cifra declaram matricele pentru cele 10 cifre si inca doua matrice pentru cazurile speciale (zero si sapte cand lipseste pixelul de pe pozitia 0x4).Dupa ce verificam daca ne aflam sau nu intr-un caz specia,l salvam apelurile functiei dif pentru fiecare dintre cele 10 matrice de cifre. Functia returneaza pozitia minimului acestui vector, care este cifra cu care portiunea din matricea principala seamana cel mai mult.
	In main deschidem fisierul de input si citim din el datele de intrare necesare taskului, respectiv numele imaginii.Construim apoi o copie a matricei de pixeli, insa cu elemente de tip char, pe care o vom parcurge cautand pixeli colorati. Atunci cand se gaseste un pixel colorat se apeleaza functia cifra pentru pozitia pixelului (i,j) si se verifica cu care cifra seamana cel mai mult. Cifrele returnate de apelul functiei cifra sunt salvate intr-un vector, pe care il vom printa la finalul parcurgerii in fisierul de output.
